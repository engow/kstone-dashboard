# Kstone Dashboard

Kstone Dashboard is a web-based UI for managing Kstone. It can help users to manage etcd based kstone and enjoy monitoring, inspection, visualization, backup and other features.

![Dashboard UI cluster page](docs/images/dashboard-ui.png)

## Getting Started

### Install

Please read [detailed install document](https://github.com/tkestack/kstone/blob/master/charts/install.md),
You can quickly install kstone through HELM.

### Develope

``` shell
yarn
yarn start
```

### Build

``` shell
make build
```

### Push Image

``` shell
make push
```
